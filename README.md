# Gesture Controlled Drone using ESP32

## Overview
This project is a hand gesture-controlled drone using ESP32. It uses sensor data and wireless communication to control drone movements in real-time.

## Components Used
- ESP32
- MPU6050 / Accelerometer
- NRF24L01 / WiFi module
- Motors and propellers

## Features
- Gesture-based control
- Wireless communication
- Real-time response
